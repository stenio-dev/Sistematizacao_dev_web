# 🏥 Clínica Aura - Portal Institucional

> Projeto desenvolvido para a Sistematização da disciplina de Desenvolvimento Web do curso de ADS do UniCEUB.

Este projeto consiste em uma solução digital moderna para uma clínica de saúde, focada em proporcionar uma experiência de usuário (UX) intuitiva, sóbria e profissional. A interface foi construída seguindo princípios de design "clean", utilizando tons pastéis e cinzas para transmitir confiança e higiene.

## 🚀 Tecnologias Utilizadas
Para garantir responsividade, performance e manutenibilidade, foram utilizadas as seguintes tecnologias:
*   **HTML5:** Estruturação semântica de múltiplas páginas.
*   **CSS3:** Estilização personalizada com paleta de cores sóbria.
*   **Bootstrap 5:** Framework para layout responsivo (Mobile First).
*   **JavaScript (Vanilla):** Lógica de validação de formulários e efeitos de interface.
*   **Google Fonts:** Tipografia profissional (Poppins).

---

## 📌 Requisitos do Projeto (Marco 1)
O desenvolvimento cumpriu integralmente os seguintes requisitos obrigatórios:
- [x] **Título Customizado:** Identificação clara na aba do navegador.
- [x] **Imagens de Serviços:** Mínimo de 2 imagens de alta qualidade representando os produtos/serviços.
- [x] **Descrição Detalhada:** Texto explicativo sobre os cuidados à saúde oferecidos.
- [x] **Equipe Especializada:** Seção com 3 membros fictícios, incluindo foto, nome e cargo.
- [x] **Hierarquia de Cabeçalhos:** Uso correto de múltiplos níveis de `<h1>`, `<h2>` e `<h3>`.
- [x] **Formulário de Contato:** Interface estática com campos de Nome, E-mail, Cidade e Estado.
